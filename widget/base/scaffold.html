# Scaffold

# 关键词（scaffold）

简介

Scaffold 实现了基本的纸墨设计布局结构。在纸墨设计中定义的单个界面上的各种布局元素，在 Scaffold 中都有支持，比如 左边栏（Drawers）、snack bars、以及 bottom sheets。

  this.bottomSheet,
  this.backgroundColor,
  this.resizeToAvoidBottomPadding,
  this.resizeToAvoidBottomInset,
  this.primary = true,
  this.drawerDragStartBehavior = DragStartBehavior.down,
})

# 2.1 属性解析

appBar：显示在界面顶部的一个 AppBar

body：当前界面所显示的主要内容

floatingActionButton：在 Material 中定义的一个功能按钮。

floatingActionButtonLocation：悬浮按钮位置

floatingActionButtonAnimator：悬浮按钮出现/消失动画

persistentFooterButtons：固定在下方显示的按钮。

drawer：侧边栏控件

bottomNavigationBar：显示在底部的导航栏按钮栏。
